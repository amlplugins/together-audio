# @amlplugins/together-audio

Native `together-ai` SDK re-exported under the `@amlplugins` namespace.

> Together Audio — Cartesia Sonic TTS with voice presets and language steering.

## Install

```bash
npm install @amlplugins/together-audio
```

## Use

```ts
import { /* named exports from together-ai */ } from "@amlplugins/together-audio";
// or
import sdk from "@amlplugins/together-audio";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
